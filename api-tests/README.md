# API tests for Customers App

## Before start

### Please install

- nvm (https://github.com/nvm-sh/nvm)

### Before first run
Open a terminal, go to the root of this repo and:
- `nvm use`
- `npm install`

**Important:** Tests require 'backend' application from 'Hopin-Exam-QE' to be up and running.

### Running tests

- `npm run test:api`
